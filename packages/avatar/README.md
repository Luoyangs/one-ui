## Avatar
a brief description of the component

### Props
| Property    | Description              | Type   | Accepted Values | Default |
| ----------- | ------------------------ | ------ | --------------- | ------- |
| size | the size of avatar | string | small、 normal、 large、 x-large | normal |
| icon | support one-icon | string |  |  |
| src | support basic image | string |  |  |
| circle | support circle style of avatar | boolean |  | false |
